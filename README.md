# GitBook Plugin Theme for Vue.js Components. Without Ads.

> This project forked by [official Vue.js organization](https://github.com/vuejs/gitbook-plugin-theme-vuejs) repository.

### Installation

``` js
"plugins": [
    "theme-vuejs-2"
]
```
